## Barter
###### Be Neighborly

Barter is a ruby on rails application built to facilitate skill based bartering.  It features map based location filtering, devise with omniauth, admin dashboard.

#### Barter
Barter is built in Rails 4 with postgresql 9.4 and includes seed data.
